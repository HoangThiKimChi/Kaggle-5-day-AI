# Vocabulary by Topic — Từ vựng theo chủ đề
> Tổng hợp từ: High-scoring IELTS Writing Model Answers (song ngữ Việt-Anh), Pauline Cullen (Cambridge Vocabulary for IELTS), Mindset L2/L3, Cambridge IELTS 14+19
> Dùng cho: `enrich_vocabulary` tool
> Cập nhật: 12 topics × 20-25 từ

---

## Hướng dẫn sử dụng

Khi user viết từ đơn điệu (ví dụ dùng "good" 3 lần), `enrich_vocabulary` tool gợi ý từ đồng nghĩa phù hợp với context học thuật.

**Nguyên tắc:** Dùng synonyms và paraphrase để tránh lặp từ — yếu tố quan trọng trong band Lexical Resource.

| good | beneficial, advantageous, positive, effective, valuable, worthwhile |

| big / large | significant, substantial, considerable, enormous, extensive |
| small | minor, limited, marginal, negligible, modest |
| show | demonstrate, reveal, indicate, highlight, illustrate, suggest |
| use | utilise, employ, apply, implement, adopt |
| think | believe, argue, contend, maintain, suggest, assert |
| say | state, claim, assert, argue, point out, highlight |
| problem | issue, challenge, concern, drawback, obstacle, difficulty |
| help | assist, support, enable, facilitate, contribute to |
| important | significant, crucial, vital, essential, fundamental, critical |
| change | alter, transform, modify, shift, reshape, revolutionise |
| increase | rise, grow, expand, surge, escalate, soar, improve |
