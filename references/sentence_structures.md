## 1. Nguyên tắc (Sam McCarter)

> "What the IELTS Writing test aims to test is not **how many words you know** but **how well you can use the words you already know**."

**Mục tiêu:** Biến đổi câu đơn điệu (S+V+O) thành câu phức tạp hơn mà giữ nguyên nghĩa — gọi là **sentence transformation for flexibility**.

## 2. Transformation Patterns — Biến đổi câu (Sam McCarter)

Mỗi cặp bên dưới có **cùng nghĩa**, khác cấu trúc. Dùng để tránh lặp lại cùng 1 kiểu câu.

### Pattern 1: Despite / In spite of
```
Original:  "I have some objections to the policy, but I support it."
Transform: "Despite having some objections to the policy, I support it."
Transform: "In spite of my objections, I support the policy."
```

### Pattern 2: Not having / Without
```
Original:  "The venture was not planned carefully enough, so it failed."
Transform: "Not having planned the venture carefully enough, it failed."
Transform: "Without careful planning, the venture was doomed to fail."
```

### Pattern 3: With + noun phrase
```
Original:  "More people are using the system, so it will fall apart."
Transform: "With more people using the system, it is likely to collapse."
```

### Pattern 4: Having + past participle
```
Original:  "The law has been relaxed, so we can focus on more serious matters."
Transform: "Having relaxed the law, we can now focus on more serious matters."
```

### Pattern 5: Much + as / Although (concession)
```
Original:  "In spite of having objections to the tax, I support it."
Transform: "Much as I object to the tax, I support the policy overall."
```

### Pattern 6: So/And (result)
```
Original:  "The system will deteriorate rapidly, thus delaying people further."
Transform: "The system will deteriorate rapidly, so/and people will be further delayed."
```

### Pattern 7: Otherwise (condition + consequence)
```
Original:  "If the government does not recruit more police, the crime rate will increase."
Transform: "The government must recruit more police officers; otherwise, the crime rate will increase."
```

### Pattern 8: Instead (substitution)
```
Original:  "The underlying cause needs to be addressed rather than tackling the result."
Transform: "Instead of tackling the result, the underlying cause needs to be addressed."
```

### Pattern 9: Resulting in / Leading to (cause-effect)
```
Original:  "The proposal was rushed through and the number of social problems increased."
Transform: "The proposal was rushed through, resulting in an increase in social problems."
```

## 3. S+V+O Variations — Biến câu đơn giản thành câu phức tạp

### Câu đơn giản (A2):
> "Technology is good. It saves time. People can shop online."

### Câu phức tạp hơn (B1-B2):
> "Technology offers significant benefits, particularly in terms of saving time. For example, people can now shop online instead of travelling to physical stores, which reduces both time and transportation costs."

**Các cách nối và mở rộng:**

| Chức năng | Cấu trúc | Ví dụ |
|---|---|---|
| Thêm ý | S + V + O, **and** S + V + O | "It saves time and reduces costs." |
| Tương phản | S + V + O, **but** S + V + O | "It is convenient, but it can cause addiction." |
| Nguyên nhân | S + V + O **because** S + V + O | "People work from home because it is more flexible." |
| Kết quả | S + V + O, **so** S + V + O | "The internet is accessible everywhere, so people can work remotely." |
| Điều kiện | **If** S + V, S + V + O | "If the government invests in education, the economy will improve." |
| Nhượng bộ | **Although** S + V + O, S + V + O | "Although it is expensive, the benefits outweigh the costs." |
| Thời gian | **When/While** S + V, S + V | "While studying, students should also exercise regularly." |

## 4. Cấu trúc câu cho từng phần bài essay

### Introduction — Paraphrase đề:
```
Passive voice:    "It has been suggested that..."
Noun phrase:      "The issue of [topic] has become increasingly important."
Cleft sentence:   "It is [topic] that has attracted considerable attention recently."
There + be:       "There is growing debate about whether..."
```

### Body — Topic sentences:
```
"One of the main advantages/disadvantages of [topic] is..."
"The most significant benefit is..."
"One problem with [topic] is..."
"The principal issue with [topic] is..."
"Another key consideration is..."
```

### Body — Supporting ideas:
```
"This is because..."            → giải thích lý do
"As a result,"                  → kết quả
"For example, in Vietnam..."     → ví dụ cụ thể
"This means that..."            → hệ quả
"Not only that, but..."         → thêm điểm
```

### Conclusion:
```
"In conclusion, while [acknowledge opposing view], I believe [your position]."
"Overall, the evidence suggests that [summary]."
"Weighing up both sides, I would argue that [conclusion]."
```

## 5. Câu mẫu theo dạng đề

### Opinion essay:
| Phần | Mẫu câu |
|---|---|
| Intro | "It is widely believed that [paraphrase]. I strongly agree/disagree with this view for two main reasons." |
| Body topic | "The most compelling reason to support this view is that [idea]." |
| Body evidence | "Evidence for this can be seen in [specific example]." |
| Conclusion | "In conclusion, I firmly believe that [restate position] because [brief summary]." |

### Discussion essay:
| Phần | Mẫu câu |
|---|---|
| Intro | "There is considerable debate about [topic]. While some argue that [View A], others contend that [View B]. This essay will examine both perspectives." |
| Body 1 | "Those who support [View A] argue that [reason]. They point to [example] as evidence." |
| Body 2 | "On the other hand, proponents of [View B] believe that [reason]. For instance, [example]." |
| Conclusion | "In conclusion, although both sides have merit, I believe [your position] is more convincing because [reason]." |

### Adv/Dis essay:
| Phần | Mẫu câu |
|---|---|
| Intro | "[Topic] has become increasingly common in recent years. This essay will examine the advantages and disadvantages of this trend." |
| Body 1 | "There are several clear advantages to [topic]. Firstly, [main advantage], which means that [effect]. For example, [specific case]." |
| Body 2 | "However, [topic] also has significant drawbacks. The most serious is [main disadvantage], as [explanation]. Moreover, [second disadvantage]." |
| Conclusion | "In conclusion, while [topic] offers [main advantage], the disadvantages — particularly [main disadvantage] — cannot be overlooked. Overall, [your final position]." |

## 6. Lỗi cấu trúc câu thường gặp (A2-B1)

| Lỗi | Sai | Đúng |
|---|---|---|
| Lặp S+V+O | "Tech is good. Tech saves time. Tech is useful." | "Technology offers numerous benefits, including time-saving capabilities and improved convenience." |
| Fragment | "Because it saves time." | "People prefer online shopping because it saves time." |
| Run-on | "People work at home it is flexible they save time." | "People work at home because it is flexible, which allows them to save time on commuting." |
| Wrong connector | "Technology is good but it saves time." | "Technology is beneficial, and it helps people save time." |
